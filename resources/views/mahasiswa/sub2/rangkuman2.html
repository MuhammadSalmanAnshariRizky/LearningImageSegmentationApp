Rangkuman

1. Pengertian Segmentasi Citra
  • Segmentasi citra adalah proses membagi citra R menjadi beberapa wilayah (region): R₁, R₂, ..., Rₙ
  • Tujuan: mempermudah analisis dengan membagi citra menjadi bagian bermakna
  • Dilakukan pada ruang spasial (x, y)
2. Aturan Segmentasi Citra

Misalkan citra dibagi menjadi R₁, R₂, ..., Rₙ, maka harus memenuhi:

(1) Cakupan Seluruh Citra
  • R = ⋃ (i = 1 s.d. n) Rᵢ
  • Semua region jika digabung membentuk citra asli
  • Tidak ada piksel yang tertinggal
(2) Keterhubungan (Connected Set)
  • Rᵢ adalah connected set
  • Setiap region harus terhubung
  • Piksel dalam satu region tidak boleh terpisah
(3) Tidak Tumpang Tindih
  • Rᵢ ∩ Rⱼ = ∅, untuk i ≠ j
  • Satu piksel hanya boleh berada pada satu region
  • Tidak boleh ada irisan antar region
(4) Memenuhi Predikat (Kriteria)
  • Q(Rᵢ) = TRUE
  • Setiap region harus memenuhi kriteria tertentu
  • Contoh kriteria (Q):
    • Intensitas sama
    • Warna sama
    • Tekstur sama
    • Standar deviasi
(5) Region Bertetangga Harus Berbeda
  • Q(Rᵢ ∪ Rⱼ) = FALSE, untuk region bertetangga
  • Dua region bertetangga harus memiliki karakteristik berbeda
  • Jika digabung masih memenuhi Q → sebenarnya satu region
3. Konsep Penting
  • Segmentasi = pembagian citra berdasarkan kesamaan karakteristik
  • Setiap region harus:
    • Homogen (seragam)
    • Terhubung
    • Tidak overlap
  • Jika satu region tidak seragam → segmentasi tidak valid
4. Metode Segmentasi Citra
a. Edge-Based Segmentation
  • Berdasarkan perubahan intensitas yang tajam (tepi)
  • Digunakan untuk mendeteksi batas objek
  • Cocok untuk citra dengan kontras tinggi
  • Kelemahan: sensitif terhadap noise dan tekstur
b. Region-Based Segmentation
  • Berdasarkan kesamaan nilai piksel
  • Mengelompokkan piksel menjadi wilayah homogen
  • Menggunakan kriteria seperti:
    • Intensitas
    • Tekstur
    • Standar deviasi
  • Cocok untuk citra dengan banyak tekstur
