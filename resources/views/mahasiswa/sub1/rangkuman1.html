Rangkuman

1. Pengertian Citra Digital
  • Representasi visual dunia nyata dalam bentuk digital
  • Dapat diolah oleh komputer
  • Tersusun dari elemen kecil bernama piksel (pixel)
  • Setiap piksel memiliki:
    • Koordinat (x, y)
    • Nilai intensitas (kecerahan/warna)
  • Representasi matematis: f(x, y)
2. Representasi Citra
  • Berbentuk matriks M × N
    • M = jumlah baris
    • N = jumlah kolom
  • Koordinat dimulai dari (0,0) (pojok kiri atas)
  • Setiap elemen matriks = nilai piksel
  • Bersifat diskrit (hasil discretization)
3. Jenis-Jenis Citra
a. Citra Grayscale
  • 1 kanal (intensitas saja)
  • Tidak memiliki warna
  • Rentang nilai (8-bit): 0 – 255
    • 0 = hitam
    • 255 = putih
  • Lebih sederhana untuk pengolahan
b. Citra Biner
  • Hanya 2 nilai:
    • 0 = hitam
    • 1 = putih
  • Menggunakan 1 bit/piksel
  • Lebih hemat penyimpanan
  • Digunakan pada:
    • Scan dokumen
    • Fax
    • Grafik sederhana
c. Citra RGB
  • 3 kanal warna:
    • Red (R)
    • Green (G)
    • Blue (B)
  • Tiap kanal: 0 – 255
  • Total: 24-bit/piksel
  • Contoh:
    • (255,0,0) = merah
    • (0,255,0) = hijau
    • (0,0,255) = biru
    • (255,255,255) = putih
4. Bit Depth
  • Jumlah bit untuk menyimpan nilai piksel
  • Rumus kemungkinan nilai: 2k
  • Contoh:
    • 1 bit → 2 nilai
    • 8 bit → 256 nilai
5. Konversi Citra
a. RGB → Grayscale
  • Tujuan: menyederhanakan pengolahan
  • Metode:
    • Lightness
    • Average
    • Luminosity
b. Grayscale → Biner (Thresholding)
  • Menggunakan nilai ambang T
  • Aturan:
    • f(x,y) > T → putih (1)
    • f(x,y) ≤ T → hitam (0)
